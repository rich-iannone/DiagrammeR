# Widget output function for use in Shiny

Widget output function for use in Shiny

## Usage

``` r
DiagrammeROutput(outputId, width = "100%", height = "auto")
```

## Arguments

- outputId:

  Output variable to read from

- width:

  A valid CSS unit for the width or a number, which will be coerced to a
  string and have `px` appended.

- height:

  A valid CSS unit for the height or a number, which will be coerced to
  a string and have `px` appended.
