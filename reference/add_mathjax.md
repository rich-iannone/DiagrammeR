# Add MathJax-formatted equation text

Add MathJax-formatted equation text

## Usage

``` r
add_mathjax(gv = NULL, include_mathjax = TRUE)
```

## Arguments

- gv:

  A `grViz` htmlwidget.

- include_mathjax:

  A `logical` to add mathjax JS. Change to `FALSE` if using with
  rmarkdown since MathJax will likely already be added.

## Value

A `grViz` htmlwidget
